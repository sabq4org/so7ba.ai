# 📊 Infographic Pro — مصمم الإنفوجرافيك

## الهدف

تحويل أي **بيانات أو معلومات أو إحصائيات** إلى إنفوجرافيك احترافي جاهز للنشر.

---

## المنهجية

### 1. تحليل المحتوى
- حدد نوع البيانات: أرقام / مقارنة / تسلسل زمني / تصنيف / خطوات
- اختر التخطيط الأنسب:
  - **هرمي** → للترتيب حسب الأهمية
  - **مقارنة** → عمودين متقابلين
  - **تسلسل زمني** → Timeline
  - **إحصائي** → أرقام كبيرة + أيقونات
  - **خطوات** → 1-2-3 تدفق
  - **خريطة ذهنية** → للمفاهيم المتشعبة

### 2. التصميم (HTML + CSS)
⚠️ **لا تستخدم توليد صور AI للإنفوجرافيك العربي** — النتائج ضعيفة جداً.

الطريقة المعتمدة:
1. أنشئ ملف HTML احترافي مع CSS مدمج
2. استخدم:
   - خط `Tajawal` أو `Cairo` من Google Fonts
   - اتجاه RTL
   - ألوان متناسقة (palette محددة)
   - أيقونات SVG أو Emoji
   - أبعاد مناسبة للمنصة (1080x1920 للستوري، 1200x630 لتويتر، 1080x1080 مربع)
3. خذ screenshot بالمتصفح

### 3. معايير الجودة
- [ ] النص واضح ومقروء (حجم 16px+)
- [ ] التباين كافي (WCAG AA)
- [ ] التنسيق متوازن بصرياً
- [ ] الأرقام بارزة وكبيرة
- [ ] الشعار/المصدر موجود
- [ ] الأبعاد مناسبة للمنصة المستهدفة

### 4. أنماط الألوان المعتمدة
```
سبق:     #1a73e8 (أزرق) + #ffffff + #f8f9fa
كبسولة:  #00c853 (أخضر صحي) + #ffffff + #f1f8e9
عاجل:    #d32f2f (أحمر) + #ffffff + #fff3f3
محايد:   #1e293b (كحلي) + #f8fafc + #e2e8f0
```

## التنفيذ

```
1. اكتب HTML كامل مع CSS inline
2. احفظه في /tmp/infographic_TIMESTAMP.html
3. افتحه بالمتصفح (canvas أو browser)
4. خذ screenshot بصيغة PNG
5. أرسله للمستخدم
```

## أمثلة استخدام

- "سوّ لي إنفوجرافيك عن إحصائيات رمضان"
- "حوّل هالأرقام لتصميم بصري"
- "ملخص مرئي لنتائج الدوري"
- "إنفوجرافيك مقارنة بين X و Y"

## ملاحظات
- الإنفوجرافيك العربي = HTML فقط (مو AI image generation)
- دائماً RTL
- دائماً خط عربي واضح
- الأبعاد حسب المنصة المستهدفة
