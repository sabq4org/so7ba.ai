سياسة الخصوصية

آخر تحديث: 1 مارس 2026

1. البيانات التي نجمعها

  • التسجيلات الصوتية: تُحفظ محلياً على جهازك فقط ولا تُرفع لخوادمنا.
  • النصوص المفرّغة: تُعالج عبر OpenAI Whisper API لتحويل الصوت إلى نص، ثم تُحفظ محلياً.
  • المصادر وجهات الاتصال: تُحفظ محلياً وتُزامن عبر iCloud الخاص بك (إن فُعّل).

2. مشاركة البيانات

  • يُرسل الصوت إلى OpenAI API لغرض التفريغ فقط — لا يُحفظ على خوادمهم بعد المعالجة.
  • لا نشارك أي بيانات مع أطراف ثالثة أخرى.
  • لا نستخدم بياناتك لأغراض إعلانية.

3. التخزين والأمان

  • جميع البيانات مخزنة محلياً على جهازك.
  • المزامنة تتم عبر iCloud الخاص بحسابك (مشفّرة).
  • مفتاح API يُخزّن محلياً على الجهاز فقط.

4. الأذونات المطلوبة

  • الميكروفون: لتسجيل المقابلات والتصريحات.
  • الشبكة: لإرسال الصوت إلى خدمة التفريغ.

5. حقوقك

  • يمكنك حذف جميع بياناتك بحذف التطبيق.
  • يمكنك إيقاف مزامنة iCloud من إعدادات الجهاز.
  • يمكنك حذف أي مقابلة أو مصدر من داخل التطبيق.
