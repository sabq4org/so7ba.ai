### مشاكل تم حلها (8 مشاكل):
1. صلاحيات ملفات (403) → `chmod -R 644`
2. nginx SPA routing (404) → `try_files $uri $uri/ /index.html`
3. API routes غلط (`/admin/login` → `/admin/auth/login`) — عدّلنا `index.ts`
4. Response format (`data.data.accessToken` → `data.token`) — عدّلنا `useAuth.tsx` + `client.ts`
5. CORS — أضفنا IP السيرفر في `index.ts`
6. OpenSSL مفقود في Alpine → `apk add openssl openssl-dev` في Dockerfile
7. جداول ناقصة بالداتابيز (`AdminSession`, `AuditLog`, `updatedAt`) → `prisma db push --accept-data-loss`
8. Redirect loop بصفحة الدخول → `if (window.location.pathname !== '/login')`

### Dockerfile نهائي:
