# أفكار جرعة — 27 فبراير 2026

## الرؤية الكبيرة
جرعة = منصة إخبارية اجتماعية ذكية — مزيج بين أخبار + تغريدات + محتوى مؤثرين بفيد مخصص.

## الفيد الذكي (الصفحة الرئيسية)
- 3 طبقات محتوى ممزوجة: أخبار (RSS) + نقاشات (تغريدات) + محتوى مؤثرين
- النسبة: 50% أخبار + 25% نقاشات + 25% محتوى (تتغير حسب المستخدم)
- بطاقات مختلفة لكل نوع (خبر = صورة+عنوان، تغريدة = تفاعلية، مؤثر = مميزة)

## التخصيص الذكي
- Onboarding: اختيار اهتمامات + مؤثرين + مصادر
- تعلم تلقائي: وقت القراءة + إعجابات + نقرات + تجاهل + متابعات
- كل مستخدم يشوف فيد مختلف
- خوارزمية: relevance 40% + recency 30% + engagement 20% + diversity 10%

## Endpoints جديدة مطلوبة
- GET /api/feed — الفيد المخصص
- GET /api/feed/trending — الأكثر تداولاً
- GET /api/creators — المؤثرين
- POST /api/user/interests — حفظ اهتمامات
- POST /api/user/follow — متابعة
- POST /api/user/interaction — تسجيل تفاعل

## جداول DB جديدة
Creator, CreatorContent, UserInterest, UserFollow, UserInteraction, FeedItem

## شاشات التطبيق
