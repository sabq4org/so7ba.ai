### Architecture Decisions
- Two-layer data: `PantryItem` (permanent catalog) → `ShoppingItem` (per-trip copy)
- `planningItems` ≠ `activeSession.items` (no shared array between tabs)
- `stepValue(for:)`: countable units step=1.0, weight/volume step=0.5
- Prebuilt catalog (local, offline) + AI populate as UI wrapper
- 5 tabs: 📦 مخزني / 🛒 سلّتي / ✅ التسوق / 🕐 السجل / ⚙️ الإعدادات
