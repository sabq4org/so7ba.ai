# بصيرة (Baseera) — تطبيق iOS

## ما هو المشروع؟
منصة ذكاء إعلامي سعودية تستهدف فرق التواصل الحكومي والعلاقات العامة. تقرأ المشهد الإعلامي، تحلل المشاعر، تكتشف الأزمات مبكراً، وتقدم توصيات لصانع القرار.

## الجمهور المستهدف
مدراء التواصل الحكومي، فرق العلاقات العامة، رؤساء التحرير — أشخاص يحتاجون يعرفون "وش يصير الحين" بثواني.

## التقنيات
- SwiftUI — iOS 16+
- Dark theme فقط
- عربي كامل (RTL)
- لا مكتبات خارجية

## الباكند
API جاهز: `http://165.227.57.44:8000`

### الـ Endpoints
```
GET /api/dashboard        ← كل بيانات المشهد (endpoint رئيسي)
GET /api/news/saudi       ← أخبار سعودية
GET /api/news/world       ← أخبار عالمية
GET /api/crisis/scan      ← الأزمات النشطة
GET /api/crisis/{id}/report ← تقرير أزمة ذكي بالـ AI
GET /api/sentiment/overview ← نظرة عامة على المشاعر
GET /api/sentiment/heatmap  ← خريطة حرارة المشاعر
GET /api/sentiment/by-topic ← مشاعر حسب الموضوع
GET /api/trends/hashtags    ← الهاشتاقات الرائجة
GET /api/early-warnings     ← إنذارات مبكرة
GET /api/pulse?hours=48     ← نبض المشهد (timeline)
GET /api/trajectory         ← مسار المشهد (يتحسن/يتراجع)
GET /api/comparison         ← مقارنة اليوم vs أمس vs الأسبوع
GET /api/radar              ← كل الأخبار
GET /api/baseline           ← خط الأساس التاريخي
GET /health                 ← حالة السيرفر
```

## شاشات التطبيق

### 1. قراءة المشهد (الرئيسية)
أول شيء يشوفه المستخدم — إجابة سريعة على: "كيف المشهد الحين؟"
- حالة المشهد: هادئ ✅ / متوتر ⚠️ / أزمة 🔴
- درجة الأمان (0-100)
- ملخص AI بلغة بشرية
- توصية (انشر عادي / انتبه / لا تنشر)
- مسار المشهد (يتحسن ↗ / مستقر → / يتراجع ↘)
- نبض المشهد (رسم بياني زمني)
- خريطة المواضيع الساخنة
- لوحة الإنذار المبكر
- مقارنة اليوم vs أمس

### 2. الرادار
أخبار لحظية من مصادر متعددة
- تابات: 🇸🇦 سعودي | 🌍 عالمي | 𝕏 تويتر | 📡 الكل
- فلاتر حسب التصنيف (سياسة، اقتصاد، رياضة...)
- كل خبر: عنوان + مصدر + صورة + وقت + شارة مشاعر
- بحث داخل الأخبار

### 3. غرفة الأزمات
رصد وإدارة الأزمات الإعلامية
- قائمة الأزمات النشطة مع درجة الخطورة
- تفاصيل كل أزمة: تحليل AI + سيناريوهات + إجراءات مقترحة + مسودة بيان
- مؤشر المسار: يتصاعد / يهدأ / مستقر

### 4. تحليل المشاعر
فهم الرأي العام
- المشاعر الستة: غضب، خوف، اشمئزاز، حزن، سخرية، قلق
- تحليل حسب الموضوع
- خريطة حرارة زمنية
- مقارنة أخبار vs تويتر

### 5. البحث
بحث حر في الأخبار والهاشتاقات

### 6. الإعدادات
تنبيهات، مصادر، تفضيلات

## الروح العامة
- التطبيق يتكلم عربي فصيح مبسط
- التصميم احترافي وهادئ — ليس "flashy"
- المعلومة الأهم أولاً دائماً
- Empty states جميلة (مو مجرد "لا توجد بيانات")
- كل رقم له سياق (مو رقم يتيم)
